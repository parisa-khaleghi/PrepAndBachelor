# Calculator

This is a simple calculator application that allows the user to perform basic mathematical calculations. The user interface consists of a text box to display the numbers and the result, and a set of buttons to input the numeric values and the operations.

# Features

The calculator supports the following operations:
<ul>
  <li>Addition: adds the value entered to the current value in the text box.</li>
  <li>Subtraction: subtracts the value entered from the current value in the text box.</li>
  <li>Multiplication: multiplies the value entered by the current value in the text box.</li>
  <li>Division: divides the current value in the text box by the value entered.</li>
  <li>Clear: resets the text box to an empty value.</li>
<li>The calculator also includes validation for input values, ensuring that only valid numeric characters are entered into the text box.</li>
</ul>

# Usage

To use the calculator, simply click on the numeric buttons to input the values, and then click on the operation buttons to perform the desired calculations. The result will be displayed in the text box. Click on the Clear button to start a new calculation.

# Installation

To run the calculator, you will need to have the .NET Framework installed on your computer. You can download it from the Microsoft website at https://dotnet.microsoft.com/download.

Once you have installed the .NET Framework, you can run the application by opening the solution file in Visual Studio and pressing the Run button.
